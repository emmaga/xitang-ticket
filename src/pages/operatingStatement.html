Groups with summary row

Overview

Because ngTable layers it's functionality onto standard html table markup, you have the freedom to include whatever you want in that markup. In this demo we use that freedom to add a summary row to each group

ngTable directive

{{ group.value }}
{{user.country}} {{user.name}} {{user.age}} {{user.money}}
Summary: {{demo.sum(group.data, "age")}} {{demo.sum(group.data, "money")}}
Grant Total: {{demo.totalAge}} {{demo.totalMoney}}

ngTableDynamic directive

{{ group.value }} {{demo.sum(group.data, "age")}} {{demo.sum(group.data, "money")}}
{{user[col.field]}}
Grant Total: {{demo.totalAge}} {{demo.totalMoney}}
